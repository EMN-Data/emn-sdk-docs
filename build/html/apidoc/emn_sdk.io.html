Get projects accessible by this user (user known by api-key).

Parameters

class emn_sdk.io.record_vocab.Vocabulary(spec: Dict)[source]¶

Bases: object

For the EMN record metadata, provide ability to add a vocabulary of required fields, conditional on the presence of certain other field/value combinations. This allows us to basically treat some fields as specifying the “type” of record, and then check that other required fields are present for records of that type.

Note to developers: this uses the “if”/”then” features in JSON Schema Draft 7.
