parse(path)[source]¶

Parse a path, which can be a file or a directory, and return a list of records.

Parameters

path – file or directory

Returns

List of records (data and associated metadata)

Raises

  • UnknownFileType – if path leads to a file that is not known, and parser is expected to parse it instead of just ignoring it.

  • ParseError – if there is some problem processing input data
